Algunos lenguales de la web

HTML + CSS + Javascript

  • HTML 5

    • HTML 5 (Hyper Text Markup Lenguage, versión 5) es la quinta revisión importante del lenguaje básico de la World Wide Web, HTML.

    Usa la extensión .html

  • CSS 3

    • Hoja de estilo en cascada o CSS (siglas en inglés de cascading style sheets) es un lenguaje usado para definir y crear la presentación de un documento estruturado escrito en HTML

    Usa la extensión css

  • Java Script

    • Java Script (abrebiando comunmente "JS" es un lenguaje de programación interpretado utilizando principalmente en el lado del cliente (client-slide) implementando como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámicas.

    Usa la extensión .js
